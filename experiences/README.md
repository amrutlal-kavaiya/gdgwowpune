# WOW 25 Experiences

This directory contains experiences shared by attendees of the GDG WOW 25 event at MIT-WPU Pune.

## Structure

Each file contains a different perspective or experience from the event, focusing on different aspects such as:

- General Event Experience
- Speaker Sessions
- Panel Discussions
- Networking Opportunities
- Learning Experiences
- Technical Insights

Feel free to browse through these experiences to get a feel for what GDG WOW 25 was like!
