# My GDG WOW Pune Experience

At GDG WOW Pune, we learned about several important AI and machine learning concepts:

## Key Concepts Explored

- **Model Context**: Understanding how AI models maintain and utilize context during conversations
- **Protocol**: Communication standards between AI systems and applications
- **Token**: The basic units that language models process text with
- **Embeddings**: Vector representations of words or phrases that capture semantic meaning
- **VectorDB**: Database systems designed to store and query vector embeddings efficiently
- **Temperature**: A parameter controlling randomness/creativity in AI responses
- **TOP-K**: Sampling technique that considers only the K most likely next tokens
- **TOP-P (Nucleus Sampling)**: Alternative sampling technique based on cumulative probability
- **RAG (Retrieval Augmented Generation)**: Enhancing LLM responses with retrieved information from external sources
- **MCP (Model Context Protocol)**: Protocol for managing context between client and server systems
- **Host**: The system or server where AI models are deployed
- **Clients**: Applications or services that request AI processing from hosts
- **Servers**: Backend systems that process AI requests and deliver responses

These concepts form the foundation of modern AI systems and applications, enabling powerful and efficient language model implementations.
