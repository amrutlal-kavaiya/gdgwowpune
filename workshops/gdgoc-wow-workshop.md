# GDGOC WOW Workshop

![GDGOC WOW Workshop Banner](https://www.gdg.community.dev/assets/logo.svg)

## About the Workshop

The GDGOC WOW (Women in Tech and Open Source Communities) Workshop is a specialized event focused on empowering women developers through open-source contribution and community engagement. This workshop combines the strengths of Google Developer Groups (GDG) and Women in Tech initiatives to create an inclusive learning environment for participants.

## Workshop Details

- **Date**: May 15, 2025
- **Time**: 10:00 AM - 4:00 PM IST
- **Location**: Tech Hub Pune, Koregaon Park, Pune, Maharashtra
- **Registration**: [Register Here](https://gdg.community.dev/events/details/)
- **Format**: Hybrid (In-person and Virtual)

## Workshop Agenda

- **10:30 - 11:15**: Keynote: "The Power of Open Source Communities" by Priya Sharma

- Networking opportunity with speakers and other participants

- **3:00 - 3:45**: Mentoring Roundtables with Industry Experts
- **3:45 - 4:00**: Closing Remarks and Next Steps

## Prerequisites

- Basic understanding of Git and GitHub
- A laptop with Git installed
- GitHub account
- Enthusiasm to learn and contribute to open-source projects

## What You'll Learn

- How to find beginner-friendly open-source projects
- The open-source contribution workflow
- Best practices for submitting pull requests
- How to become a regular contributor to open-source projects
- Building your personal brand through open-source contributions
